# Product ideas

These documents describe four long-horizon UE Shed product directions. They share discovery,
producer/world/session identity, capability negotiation, bounded streams, and evidence, while keeping
their domain models separate.

- [Real-time actor observatory](actor-observatory.md)
- [Live camera feeds](live-camera-feeds.md)
- [Map review cameras](map-review-cameras.md)
- [Interactive gameplay scenarios](interactive-scenarios.md)

They are vision documents, not commitments to implement every mechanism exactly as first described.
Risky decisions are intentionally earned through vertical slices and engine-source verification.
