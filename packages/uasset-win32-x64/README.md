# `@ue-shed/uasset-win32-x64`

The Windows x64 native artifact used by `@ue-shed/uasset@0.1.0-rc.3`. Consumers should install the
launcher package rather than depending on this package directly.

This package contains the release-built `uasset.exe` and package metadata only. It performs no
runtime download, build, or platform detection. The executable is a read-only Unreal saved-asset
parser and is MIT licensed.
